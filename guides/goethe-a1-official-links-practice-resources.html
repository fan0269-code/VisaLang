Source map

Safe practice rule

Good A1 practice should help you speak, listen, read, and write simple German in everyday situations. It should not pretend to be the real exam, sell fake certainty, copy protected material, or train you to memorize one fixed answer.

  • Use official Goethe practice as the format reference.
  • Use original prompts for repetition and confidence.
  • Keep a weekly weak-area list instead of chasing random PDFs.
  • Check local exam format, date, and ID rules before paid prep.

Resource checklist

Resource type Useful if Reject if
Official exam page It gives current exam, venue, and practice information. It is not from Goethe or an official local venue path.
Paid course It teaches A1 basics and points back to official format. It promises a pass, fake certificate, or secret questions.
Practice PDF It is official, original, or clearly licensed. It claims to contain real current exam questions.
Tutor or school It can explain your weak areas and avoids visa promises. It says it can guarantee approval or bypass the exam.

Red flags

These are reasons to stop before paying.

"Guaranteed pass" or "guaranteed visa"

No practice resource can guarantee an exam result or immigration decision.

"Real current exam questions"

Do not use leaked or copied real exam items. Build skill with official samples and original practice instead.

No official link or checked date

If a resource talks about fees, dates, centers, or rules without a source and checked date, treat it as unstable.
