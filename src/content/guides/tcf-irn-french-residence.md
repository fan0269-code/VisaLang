## Start with the exact procedure

The current Ministry of the Interior nationality page states that, since 1 January 2026, naturalisation and reintegration by decree and declaration by marriage use a B2 oral-and-written language threshold. Language proof is one dossier condition; it does not decide eligibility, assimilation, registration of a declaration, or the outcome.

## Do not transfer this rule to residence applications

The current Service-Public residence page distinguishes covered first applications for multi-year residence cards and resident cards, with different A2 and B1 thresholds and route-specific exclusions. This does not establish one rule for every residence card, renewal, applicant, agreement, or dispensation. Select the exact card and applicant category before relying on a level.

## What the test owner confirms

France Éducation international describes TCF IRN as a four-skill language test that currently evaluates up to B2. It does not test French history, culture, or society; the civic examination is a separate procedure. The product page does not decide whether a specific dossier accepts a particular result.

## Questions for the receiving authority

- Which procedure and applicant category apply to the application?
- Is language proof required, and are exemptions or alternative documents available?
- If a TCF result is accepted, which product, level, result document, and submission timing apply?
- Which official procedure page records those requirements?

## Local execution remains dynamic

Use the current French product page, then confirm the authorised centre, session, fee, ID, cancellation, result, and document-delivery details. Do not publish or rely on one fixed retake interval: current first-party FEI material is inconsistent on that point.

## Next action

Identify the exact Ministry nationality procedure and save its current language-proof instructions before booking TCF IRN.
