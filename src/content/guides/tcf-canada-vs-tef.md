## Quick comparison

- **TEF Canada:** Run by CCI Paris Ile-de-France. More international test centres. Multiple-choice and written sections.
- **TCF Canada:** Run by France Education International. More questions per section, computer-adaptive testing in some sections.

## How to choose

Both tests give equivalent CLB scores for IRCC purposes. Pick the one with the earliest available test date and the nearest centre. Both tests are accepted for the same immigration streams, so there is no strategic advantage to one over the other.
